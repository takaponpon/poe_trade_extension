# Path of Exile 2 (PoE2) Trade Sidebar Extension

Path of Exile 2の取引サイト（公式トレードページ）の右側に、美しく使いやすい設定サイドバーを直接追加するGoogle Chrome拡張機能のベースプロジェクトです。

## 主な特徴

- **ゲームに馴染むプレミアムデザイン**: PoE2の世界観にマッチした、黒（黒曜石）と金（ゴールド・アンバー）を基調とするダークファンタジー調のUIデザイン。
- **孤立化されたスタイリング**: Shadow DOMを採用し、公式サイト側のスタイル定義と完全に分離。表示崩れの心配がありません。
- **レイアウトの自動調整 (Shifting)**: サイドバー表示時、トレードページ全体が左側にスムーズに縮小シフトするため、公式の検索フォームや検索結果がサイドバーで隠れません。
- **スマートな折りたたみ機能**: 必要に応じて、右端の `>` または `<` ボタンを押すだけで、サイドバーを瞬時に最小化できます。
- **状態の自動保存**: 折りたたみ状態、チェックボックスの値、およびテキスト入力の内容は `chrome.storage.sync` を使用して自動保存され、ページを更新しても維持されます。

## 開発環境（Google Chrome）へのインストール方法

この拡張機能は、Chromeのデベロッパーモードを使用して、ローカルのフォルダから直接読み込ませることができます。

1. **Google Chrome** を開きます。
2. アドレスバーに `chrome://extensions/` と入力してエンターキーを押し、拡張機能管理ページを開きます（または、右上メニューの「拡張機能」＞「拡張機能を管理」を選択）。
3. 画面右上にある **「デベロッパー モード」** のトグルスイッチを **オン** にします。
4. 画面左上に出現する **「パッケージ化されていない拡張機能を読み込む」**（Load unpacked）ボタンをクリックします。
5. ファイル選択ダイアログが表示されるので、このプロジェクトのルートディレクトリである以下のフォルダを選択します：
   `c:\Users\takah\Documents\repository\poe_trade_extension`
6. 拡張機能一覧に **「PoE2 Trade Sidebar Extension」** が表示されればインストール完了です！

## 動作確認方法

インストールが完了したら、以下の公式トレードサイトにアクセスしてください：

- **日本語版公式トレードサイト**: [https://jp.pathofexile.com/trade2/search/poe2](https://jp.pathofexile.com/trade2/search/poe2)
- **英語版公式トレードサイト**: [https://www.pathofexile.com/trade2/search/poe2](https://www.pathofexile.com/trade2/search/poe2)

ページが開くと、画面右側に美しくアニメーションするサイドバーが表示されます。

- **開閉チェック**: サイドバー左上の `>` ボタンを押すと折りたたまれ（幅40px）、`<` を押すと元のサイズ（250px）に戻ります。それに合わせてメインコンテンツが左右にスムーズにスライドすることを確認してください。
- **設定保存チェック**: 「Auto-Refresh」や「Sound Alerts」のトグルを切り替えて「APPLY SETTINGS」ボタンを押すと、「SAVED!」という緑色のフィードバックが表示され、値が保存されます。ページを更新しても値が維持されていることを確認できます。
